# Maps, Characters, And Plots

## Map Creation

For each map, specify:

- tactical purpose
- dimensions/aspect ratio
- terrain zones
- cover
- hazards
- secret areas
- starting positions
- lighting/weather
- no text/labels unless explicitly requested

## NPC Creation

Every NPC needs:

- want
- fear
- secret
- voice
- table line
- combat/social role
- relationship to at least one faction

## Uncle Yev Twist Rules

Twists should be unexpected but fair:

- foreshadow at least once
- connect to a player choice
- make the world feel alive
- create a new decision, not just a reveal

Bad twist: “the villain was immune all along.”

Good twist: “the villain uses the exact clever shortcut the players invented last session, because an agent familiar observed it.”

## Loot Rules

Loot should do at least one:

- reveal history
- create obligation
- unlock a non-combat option
- tempt a tradeoff
- point to a future location
